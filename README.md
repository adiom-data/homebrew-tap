# Adiom-data Tap

## How do I install these formulae?

`brew install adiom-data/tap/<formula>`

Or `brew tap adiom-data/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "adiom-data/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
